An anonymous real-time journal feed for posting raw thoughts without profiles, names, or setup friction.

Users can publish short messages instantly, react with likes, and continue the conversation through threaded replies while the feed updates live.

## Features

- **Anonymous Posting**: Share thoughts instantly without creating a public identity.
- **Real-time Feed**: New posts, reactions, and viewer activity update live.
- **Replies and Reactions**: Users can like entries, open comment threads, and add anonymous replies.

## Technology Stack

- **Frontend**: React
- **Backend**: Convex for realtime data sync
- **UI**: Tailwind CSS
- **Analytics**: Betterlytics
